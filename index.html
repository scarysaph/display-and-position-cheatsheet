Display and Positioning CSS cheatsheet

By default, a browser will render elements in a html document with no css from top to bottom and left to right, in the order they appear. Here are 5 CSS properties that change how the browser positions elements.

Position

This property changes the position of the element, using offset properties top, bottom, left and right.

Values Description
static The default position. Element block takes up the full width of parent element, prevent other elements appearing in the same horizontal space and appear on the left side of the browser.
relative Element is positioned relative to usual static position.
absolute Element is removed from the flow of the document (ignored by other elements) and positioned relative to its closest non-static parent element.
fixed Element is removed from the flow of the document(ignored by other elements) and fixed to a point on the page regardless of user scrolling.
sticky Element acts position: relative until it reaches the position defined by offset properties. Then it acts position: fixed regardless of scrolling until it reaches the end of its parent element where it will rejoin document flow.
top, bottom, right, left Offset properties that position element away from respective directions.

Display

This property dictates how much space an element block takes up.

Values Description
inline Elements have a box wrapped tightly around content, taking up only the space required to dispaly it and not requiring a new line afterwards. Element size cannot be altered using width and height. Default display for <em>, <a> and <strong>.
block Element displayed on a separate line and takes up width of page by default but its size can be adjusted. Default display for <h1> to <h6>, <p>, <div> and <footer>.
inline-block Elements can appear next to each other and height and width can be adjusted. Default display for images.

Z-Index

This property controls how far back or forwards an element appears on the page.

  • It takes integer values with the largest value appearing on top.
  • The default value is 0.
  • Used only on non-statically positioned elements.

Float

This property moves an element as far left or as far right as possible in its container. The element must have a width specified otherwise it will be assumed to be the same as its parent element and float will have no effect. It is better to use grid or flexbox.

Values Description
left Element is floated as far left as possible.
right Element is floated as far right as possible.

Clear

This property specifies how elements should behave when they bump into each other on the page. Elements with the clear property will move to the next line if they come in contact with a floated element on that side.

Values Description
left The left side of the element will not touch any other element within the same containing element.
right The right side of the element will not touch any other element within the same containing element.
both Neither side of the element will touch any other element within the same containing element.
none The element can be touched on either side.
